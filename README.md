**Self-Mail** üì®: A Simple Static Website for Sending Emails

üìñ Description
---------------
Self-Mail is a static website built using HTML, CSS, and JavaScript, designed to provide a simple and intuitive way to send emails. The website utilizes the EmailJS library to handle email sending, allowing users to input their message and recipient's email address. The website is fully responsive, ensuring a seamless user experience across various devices and screen sizes.

The website's design is minimalist and modern, featuring a linear gradient background that adds a touch of elegance to the overall aesthetic. The user interface is straightforward, with a single input field for the recipient's email address, a textarea for the message, and a send button. Upon clicking the send button, the website displays a popup message indicating whether the email was sent successfully or not.

One of the key benefits of using Self-Mail is its simplicity and ease of use. Users do not need to create an account or log in to send an email, making it an ideal solution for those who need to send a quick message without the hassle of setting up an email client. Additionally, the website's static nature ensures that it can be hosted on any server, making it a great option for developers who want to create a simple email sending application.

üìÅ Project Structure
---------------------
The project consists of the following folders and files:
* **index.html**: The main HTML file that contains the website's structure and content.
